# Led Panel Manager (React)

Sistema moderno para gerenciamento de painéis LED, migrado para React + Vite com backend Node.js.

## 🚀 Deploy Online

**URL de Produção**: [Será gerada pelo Railway após deploy]

## 🏠 Desenvolvimento Local

### Opção 1: Automática (Recomendada)
```bash
# Execute o arquivo start-all.bat
start-all.bat
```

### Opção 2: Manual
```bash
# Terminal 1 - Backend
cd backend
npm install
npm start

# Terminal 2 - Frontend  
npm install
npm run dev
```

## 📦 Tecnologias

- **Frontend**: React 18 + Vite + CSS Moderno
- **Backend**: Node.js + Express + JSON Storage
- **PDF**: @react-pdf/renderer
- **Deploy**: Railway (Full-stack)

## 🎯 Funcionalidades

- ✅ Cadastro de gabinetes LED
- ✅ Gerenciamento de projetos
- ✅ Criação/edição de painéis
- ✅ Cálculos elétricos avançados
- ✅ Relatórios PDF com grid técnico
- ✅ Agenda de eventos
- ✅ Interface responsiva moderna
